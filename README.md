# iogame_dwec25
juego IO de 2048 competitivo

Para jugar se puede acceder al desplegamiento en Vercel: https://iogamedwec25samuel.vercel.app/




--Normas de juego--

· Jugador 2 pulsa en una casilla del tablero del jugador 1

· Jugador 1 mueve el tablero usando wasd

· Jugador 1 pulsa en una casilla del tablero del jugador 2

· Jugador 2 mueve el tablero usando wasd

· Repetir estos pasos hasta que un jugador no pueda mover el tablero
